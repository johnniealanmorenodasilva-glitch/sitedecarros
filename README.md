# Exotic Garage - Catálogo de Carros Premium

Este é um projeto de um site moderno e premium de catálogo de carros, construído com Vanilla HTML, CSS e JavaScript. Ele exibe uma vitrine de modelos com design inovador, incluindo glassmorphism, micro-animações e foco em usabilidade.

## Como Executar Localmente com Docker

Este projeto está pronto para ser containerizado e servido através de um servidor Nginx extremamente rápido.

1. **Construir e Iniciar o Container**
   ```bash
   docker-compose up -d --build
   ```

2. **Acessar o Site**
   Abra seu navegador e acesse: [http://localhost:8080](http://localhost:8080)

3. **Parar o Container**
   ```bash
   docker-compose down
   ```

## Tecnologias Utilizadas
- **HTML5** (Estrutura semântica)
- **CSS3** (Estilização premium com variáveis e animações)
- **JavaScript (ES6+)** (Lógica de exibição e interações via Módulos)
- **Docker & Nginx** (Containerização e Servidor Web)
